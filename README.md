# Press Release Generator

PR Times向けプレスリリース自動生成ツール。esse-sense社のリリース形式をベースに、企業向けプレスリリースを対話形式で作成します。

## 対応リリース種類

- **サービスリリース** - 新規プロダクト・機能のリリース
- **提携・協業** - 業務提携、共同研究、MOU締結等
- **資金調達完了** - シード、シリーズA等の資金調達
- **イベント開催** - カンファレンス、セミナー等
- **サービスアップデート** - 既存プロダクトの大幅改善
- **受賞・認定** - 表彰、認証取得等

## 使い方

### ウェブアプリ（推奨）

```bash
streamlit run app.py
```

ブラウザでフォームに入力するだけで、リアルタイムにプレスリリースをプレビュー・ダウンロードできます。

### CLIモード

```bash
python main.py                  # 対話モード
python main.py --type service   # リリース種類を事前指定
python main.py --output pr.md   # ファイルに出力
python main.py --profile name   # 保存済みプロフィールを使用
```

### プロフィール機能

会社情報（社名、所在地、代表者名、連絡先等）を一度入力すると、プロフィールとして保存できます。次回以降はプロフィール名を指定するだけで会社情報の入力をスキップできます。

## 出力形式

Markdown形式で出力されます。PR Timesへの入稿時は以下の点にご注意ください：

- PR Timesの本文エディタにMarkdownをそのまま貼り付けると、見出し等が反映されます
- 画像は3〜5枚以上を準備してください（PR Times推奨）
- トップ画像はSNSシェア時のサムネイルとして使用されます

## PR Times最適化ポイント

本ツールは以下のPR Timesベストプラクティスに準拠しています：

- **逆三角形構造**: タイトル → サブタイトル → リード文 → 本文で情報量を段階的に増やす
- **5W2H対応**: リード文にWho/What/When/Where/Why/How/How muchを含む
- **数値重視**: 具体的な数値データを含めるプロンプト設計
- **会社概要ボイラープレート**: 末尾に定型の会社情報・問い合わせ先を自動付与

## 必要環境

- Python 3.10以上
- Streamlit (`pip install streamlit`)
